The Power of Presentation

A webpage without CSS is like wine without a wine glass. Sure, it's still wine, and it's still possible for you to consume it, but it's so much more enjoyable to consume it with style.

This lab is a technical one, so our focus is going to be on how to use the technology of CSS to display the elements of a webpage. CSS stands for Cascading Style Sheets. They're called a style sheets because they are documents that contain instructions for the stylistic presentation of the HTML. They're said to be "cascading" because of the way the styling rules "cascade" according to a priority schedule to all the elements in the HTML.

For Lab 3, the HTML content is this page itself. Your job is to create the CSS. The styling requirements are given in the bulleted list below...

Requirements and Grading

To get started with this lab, you will need to download the content and put it into the correct subfolders, which you will create. The two subfolders will be called css and images. The content you need to dowload is available on the Assigments page of D2L: specimens.png, laboratorywallpaper.jpg, and main.css.

The following requirements should all be accomplished by modifying the main.css stylesheet. Make sure you it exists in a folder called css, which should be in the same folder as your index.html file and your images folder. Do not modify the index.html document.

  • 1 point:
    All level 1 header text, <h1>, should appear as sans-serif at 50 pixels high and have a background color of rgb(60,40,30). The text in the header 1 element should also be centered.
  • 1 point:
    All level 2 header text, <h2>, should appear as serif text that is 30 pixels high. The text should be dark red, italicized, styled in the small-caps font variant, and have a 3 pixel spacing between each character.
  • 1 point:
    The specimens.png image should be centered and have a 10 pixel solid border that matches the header 1 background color. Make sure to store it in the /images directory so it shows up!
  • 1 point:
    The labratorywallpaper.png background image should be tiled to take up the whole page. Rename this image labBkgd.png and store it in the images subdirectory.
  • 1 point:
    The spacing between each bullet item in a list is 10 pixels.
  • 1 point:
    All paragraph text, <p>, and text in a list has a background color of #f0f0ef so that the text won't get lost in the busy background pattern.
  • 1 point:
    When the cursor is placed above a hypertext link, the text of the link should turn white, be in all uppercase letters, should not be underlined, and should have a background color of dark blue.
  • 1 point:
    Any text that is strong, <strong>, in the original html document should be dark red in color, and rendered in a italic serif font.
  • 1 point:
    The main.css file validates.
  • 1 point:
    The expected directory structure is correct.
